# 🚀 CleanPro - Premium Landing Page Template (Titan Mode)

**CleanPro** é um template de elite desenvolvido para empresas de serviços que exigem uma presença digital de alto impacto. Focado em **conversão**, **velocidade** e **estética moderna**, este projeto utiliza as tecnologias mais avançadas do ecossistema Web de 2026.

---

## 💎 Destaques do Projeto

* **⚡ Next.js 14+ (App Router):** Performance extrema e SEO otimizado nativamente.
* **🎨 "Modo Deus" (Tailwind Config):** Sistema de variáveis globais que permite mudar toda a identidade visual (cores, gradientes e arredondamentos) em segundos.
* **✨ Glassmorphism UI:** Efeitos de vidro fosco e profundidade visual de última geração.
* **🚀 Framer Motion:** Animações fluidas e micro-interações que elevam a experiência do utilizador.
* **📱 Mobile-First:** Design 100% responsivo, testado para máxima conversão em smartphones.
* **🔍 SEO Ready:** Configuração de metadados preparada para o ranking do Google.

---

## 🛠️ Tecnologias Utilizadas

* **Framework:** [Next.js](https://nextjs.org/)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
* **Animações:** [Framer Motion](https://www.framer.com/motion/)
* **Tipografia:** Inter (Google Fonts)

---

## ⚙️ Como Começar

1. **Instale as dependências:**
  
   npm install

   npm run dev

   Altere as cores globais no ficheiro tailwind.config.ts para adaptar o template a qualquer nicho de mercado.
